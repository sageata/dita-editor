# Provenance

`docs/public-export.json` is the exact, versioned public-export path inventory. `docs/provenance.json` binds every path in that inventory except the provenance file itself to a lowercase SHA-256 and records origin, owner or author, license basis, the exact `public-export` disposition, reviewer, review date, and approval. The verifier rejects inventory disagreement, missing, stale, duplicate-normalized, case-fold-colliding, unapproved, placeholder, unclassified, non-regular, or linked files, then materializes and scans exactly the approved set. This document explains the record; the provenance JSON intentionally does not hash itself because the final private Task 13 export manifest binds its bytes.

Authored source, tests, synthetic fixtures, scripts, configuration, and documentation were reviewed for a history-free Apache-2.0 public export. `bun.lock` contains third-party package metadata; applicable upstream licenses remain listed through the lock and `THIRD_PARTY_NOTICES.md`.
