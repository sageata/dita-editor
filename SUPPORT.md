# Support

Use [GitHub Issues](https://github.com/sageata/dita-editor/issues) for reproducible bugs and focused feature requests. Search existing issues first and include the extension version, VS Code version, desktop OS, trust state, expected behavior, and a minimal synthetic DITA sample. Remove private content and credentials.

This free preview is maintained on a best-effort basis and has no service-level agreement. Questions about DITA-OT publishing, custom schemas, or Visual Studio Code itself may be better directed to those projects. Report security concerns privately as described in [SECURITY.md](SECURITY.md).
