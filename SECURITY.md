# Security policy

## Supported version

Security fixes are provided for the latest published preview only. Version 0.1.0 is the first planned preview.

## Report a vulnerability

Do not open a public issue for a suspected vulnerability. Submit a confidential report through [GitHub private vulnerability reporting](https://github.com/sageata/dita-editor/security/advisories/new). Include affected versions, impact, reproduction steps, and a minimal proof of concept without private document content.

The maintainer will acknowledge a valid report as soon as practical, coordinate investigation and remediation, and disclose it after a fix is available. Please do not publish details before coordinated disclosure.
